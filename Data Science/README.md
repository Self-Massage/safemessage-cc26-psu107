# Data Science - SafeMessage

Folder ini berisi seluruh pekerjaan tim Data Science untuk proyek SafeMessage,
sistem deteksi phishing message Bahasa Indonesia berbasis NLP menggunakan TextCNN.

## Struktur Folder

| Folder | Deskripsi |
|--------|-----------|
| `dataset/raw/` | Dataset mentah dari sumber asli |
| `dataset/processed/` | Dataset hasil data wrangling, siap digunakan untuk training model |
| `notebook/` | Seluruh kode eksplorasi, analisis, dashboard, dan pengujian model |

## Cara Menjalankan
1. Download dataset dari sumber (lihat `Dataset/Raw/README.md`)
2. Jalankan notebook sesuai urutan di folder `Notebook/`
3. Untuk dashboard, lihat panduan di `Notebook/Dashboard/README.md`
