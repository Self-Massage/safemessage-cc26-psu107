# Dashboard

Folder ini berisi kode Streamlit untuk visualisasi EDA dan monitoring performa model SafeMessage.

## Akses Dashboard
Dashboard telah di-deploy dan dapat diakses di:
https://phishing-detection-eda.streamlit.app/

## Menjalankan Secara Lokal
1. Install dependencies:
   pip install -r requirements.txt
2. Jalankan dashboard:
   streamlit run app.py
