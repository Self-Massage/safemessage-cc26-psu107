# Notebooks

Folder ini berisi Jupyter Notebook untuk proses eksplorasi dan persiapan data proyek SafeMessage.

## Daftar Notebook

| File | Deskripsi |
|------|-----------|
| `01_data_wrangling.ipynb` | Proses gathering, assessing, dan cleaning dataset phishing Bahasa Indonesia |
